================================================================================
    MICROSOFT 基礎類庫: calendar2 專案概述
===============================================================================

應用程式精靈已為您新建此 calendar2 應用程式。此應用程式不僅演示使用 Microsoft 基礎類別的基本知識，
而且可作為編寫應用程式的起點。

此檔包含組成 calendar2 應用程式的各個檔的內容摘要。

calendar2.vcproj
    這是使用應用程式精靈生成的 VC++ 專案的主專案檔案。
    它包含有關生成檔的 Visual C++ 版本的資訊，以及有關使用應用程式精靈選擇的平臺、配置和專案功能的資訊。

calendar2.h
    這是應用程式的主要標頭檔。它包括其他專案特定的標頭檔(包括 Resource.h)，並宣告 Ccalendar2App 應用程式類別。

calendar2.cpp
    這是包含應用程式類別 Ccalendar2App 的主要應用程式原始檔案。

calendar2.rc
    這是程式使用的所有 Microsoft Windows 資源的列表。它包括 RES 子目錄中儲存的圖示、點陣圖和游標。此檔可以直接在 Microsoft Visual C++ 中進行編輯。專案資源位於 2052 中。

res\calendar2.ico
    這是用作應用程式圖示的圖示檔。此圖示包括在主要資源檔 calendar2.rc 中。

res\calendar2.rc2
    此檔包含不是由 Microsoft Visual C++ 編輯的資源。您應該將不可由資源編輯器編輯的所有資源放在此檔中。


/////////////////////////////////////////////////////////////////////////////

應用程式精靈新建一個對話方塊類別:

calendar2Dlg.h，calendar2Dlg.cpp - 對話方塊
    這些檔案包含 Ccalendar2Dlg 類別。該類別定義應用程式主對話方塊的行為。該對話方塊的範本位於 calendar2.rc 中，該檔案可以在 Microsoft Visual C++ 中進行編輯。


/////////////////////////////////////////////////////////////////////////////

其他功能:

ActiveX 控制項
    應用程式包括對使用 ActiveX 控制項的支援。

/////////////////////////////////////////////////////////////////////////////

其他標準檔:

StdAfx.h, StdAfx.cpp
    這些檔案用於生成名為 calendar2.pch 的預編譯標頭(PCH)檔和名為 StdAfx.obj 的預編譯類型檔。

Resource.h
    這是標準標頭檔，它定義新資源 ID。
    Microsoft Visual C++ 將讀取並更新此檔。

calendar2.manifest
	Windows XP 使用應用程式清單檔案描述應用程式
	對特定版本並行程式集的依賴性。載入程式使用此
	資訊從元件快取載入相應程式集或
	從應用程式載入私有資訊。應用程式清單可能作為
	與應用程式可執行檔安裝在同一資料夾中的外部 .manifest 檔包括在內以便重新發佈，
	也可能以資源的形式包括在該可執行檔中。

其他注釋:

應用程式精靈使用“TODO:”指示應新增或自訂的原始程式碼部分。

如果應用程式在共用 DLL 中使用 MFC，則將需要重新發佈 MFC DLL。如果應用程式所用與作業系統的區域設定不同，則也將必須重新發佈對應的當地語系化資源 MFC90XXX.DLL。
有關這兩個主題的詳細資訊，請參閱 MSDN 文件中有關重新發佈 Visual C++ 應用程式的部分。
